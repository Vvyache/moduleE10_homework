Vyacheslav Batalov Студент курса Fullstack-разработчик на Python

  • March 2023
    Skill Factory
    В марте 2023 года, я принял решение освоить навыки программирования. Мой выбор пал на Python, так как привлекали задачи, которые можно решить с помощью этого языка. Для обучения я обратился в online-школу SkillFactory, где были подходящие условия для обучения и интересующий меня курс Fullstack-Python. Курс заинтересовал ещё и тем, что содержит не только учебные материалы по Python для разработки серверной части приложений, но и большой блок по frontend, реализация которого осуществляется по средствам JavaScript, HTML, CSS и многих других инструментах, о которых ещё предстоит узнать.
  • March-June 2023
    Python
    В блоке курса посвященного Python, обучение началось с базовых основ языка: типы данных, условия, циклы, функции, а так же были освоены дополнительные инструменты разработчика такие как GitHub, и различные среды разработки (IDE). Тема изучения Python не могла обойти стороной и ООП, где были изучены дополнительные возможности языка и сферы его применения. Итоговыми проектами по блоку Python были консольные игры: Крестики-нолики написанные в функциональном стиле и Морской Бой написанный с объектно-ориентированной парадигмой программирования(ООП), а также Telegram-бот конвертер валют
  • June-September 2023
    Django фреймворк
    Дальнейшим погружением во Fullstack-разработку было знакомство с фреймворком Django. Начиная с самых базовых основ фреймворка таких как создание моделей, шаблонов, фильтров и форм и более глубоко погружаясь в работу backend-приложений, взаимодействия с реляционными БД (SQLite3, PostgreSQL) и взаимодействия через REST API. Также благодаря введению в HTML, появилась возможность визуализации взаимодействия серверной и клиентской части веб-приложения. Итоговым проектом по блоку Django было создание веб-приложения Новостного портала и Доски объявлений.
  • October-December 2023
    JS, HTML, CSS
    Освоение frontend началось с базовых основ JS(функции, объекты, документ, AJAX, API браузера). Итоговой работой стала реализация базового мессенжера, состоящего из клиента на JavaScript и бэкенда на Django Rest Framework. На данном этапе обучения проходит дальнейшее погружение в визуальное оформление веб-страниц, изучение расширенных возможностей CSS (flexbox, grid, @-правила и медиа-запросы). Итоговой работой по модулю Advanced CSS и станет данная веб-страница.
  • 2024
    Обучение продолжается
    Обучение программированию никогда не заканчивается и вступив на этот путь я в этом убедился. Каждый день появляется что-то новое, совершенствуются языки, разрабатываются новые технологии. Следующим этапом моего обучения уже в новом 2024 году будет React, благодаря которому я смогу ещё больше погрузиться в разработку пользовательских интерфейсов.
